# Дипломная работа Frontend - Онлайн кинотеатр

## Описание 

Web-приложение для бронирования билетов в онлайн кинотеатре. С возможностью выбора мест и сохранением QR-кода, который будет хранить информацию о бронировании. Также реализована функциональность администратора, которая позволяет добавлять залы и управлять ими. Осуществлена возможность добавления сеансов и фильмов на таймлайн с помощью `drag-and-drop`. 

__Особенности логики__

- Сеансы на текущий день фильтруются по времени (нельзя выбрать прошедшие)
- Для будущих дней отображаются все сеансы
- Бронирование сохраняется с генерацией QR-кода
- Администратор может:
  - управлять залами
  - настраивать цены
  - добавлять фильмы через drag-and-drop

__Функции__

- Бронирование и выбор мест
- Сохранение QR-кода
- Добавление залов
- Добавление сеансов и фильмов
- Открытие и закрытие залов
- Настройка конфигурации залов и цен на билеты
- Просмотр расписания
- Просмотр сеансов и фильмов

## Стек технологий

- __React__
- __Vite__
- __TypeScript__

## Зависимости 

- `react`
- `react-router-dom`
- `zustand` — управление состоянием
- `dayjs` — работа с датами
- `axios` — HTTP-запросы
- `dotenv`
- `react-qr-code` — генерация QR

__Архитектура__

- Приложение разделено на:
  - **Client** — пользовательская часть
  - **Admin** — панель администратора
- Данные загружаются через API-сервисы (`service/`)
    -  Запросы к API-сервису для получегний данных — `axios`
- Глобальное состояние — `zustand`
    - Для хранения цвета сеанса
- Работа с датами — `dayjs`

## Запуск программы

- Сборка программы: `npm run build`
- Установка зависимостей: `npm install`
- Запуск программы: `npm run dev`
- Привью программы: `npm run preview`
